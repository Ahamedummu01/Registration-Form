Registration Form

This project is a simple Registration Form built using HTML, CSS, and JavaScript.
It allows users to enter their details and submit the form.
-----------------------------------------------------------------------------------

🚀 Features

User-friendly registration form

Input fields for Name, Email, and Password

Basic form validation using JavaScript

Form automatically clears after submission

Simple and responsive design
-----------------------------------------------------------------------------------

🛠️ Technologies Used

HTML

CSS

JavaScript

------------------------------------------------------------------------------------
📂 Project Structure
Registration-Form/
│
├── index.html
├── style.css
└── script.js

-------------------------------------------------------------------------------------

📸 Preview

This page displays a registration form where users can enter their details and register.

---------------------------------------------------------------------------------------

📌 How to Use

Open the project folder.

Run the index.html file in your browser.

Enter your details in the form.

Click the Register button.
----------------------------------------------------------------------------------------

🎯 Purpose

This project was created to practice form handling and basic JavaScript DOM manipulation.
